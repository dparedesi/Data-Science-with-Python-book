# Book Details

**Title:**
Data Science with Python

**Subtitle:**
Data Analysis and Prediction Algorithms with Python

**About this book:**
Learning to make data-driven decisions is today a cross-industry requirement. In this book, you will find everything from the basics of data analysis to visualization, probability theory, inference, modeling, and machine learning. The content is designed to provide opportunities for new talents looking to train as Data Scientists. This book uses modern Python tools like pandas, scikit-learn, matplotlib, and seaborn, and includes a section on Generative AI.

**Teaser text:**
Discover how to become a Data Scientist at your own pace with updated content and real-world examples. From data analysis to prediction algorithms with machine learning. Includes chapters on Generative AI, Ethics, and modern Machine Learning workflows using Python.

**Meta description:**
data science, Python, machine learning, probability, modeling, artificial intelligence, pandas, scikit-learn, generative AI, llm
