# Contributing to Data Science with Python

Thank you for your interest in contributing to this book! Your contributions help make this resource better for everyone learning data science with Python.

## How to Contribute

### Reporting Issues

- Use [GitHub Issues](https://github.com/dparedesi/Data-Science-with-Python-book/issues) to report errors, typos, or unclear explanations
- Please include the chapter and section where you found the issue
- For code errors, include the error message and your Python version

### Suggesting Improvements

- Open an issue describing your suggestion
- Be specific about what you'd like to see changed or added

### Submitting Pull Requests

1. Fork the repository
2. Create a new branch for your changes
3. Make your changes
4. Test by running `quarto preview`
5. Submit a pull request

## Code Style

- Follow PEP 8 for Python code
- Use descriptive variable names
- Include comments for complex operations
- Keep code examples simple and educational
